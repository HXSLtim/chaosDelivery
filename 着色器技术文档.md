# 《混乱快递》着色器技术文档

> **文档版本**: v1.0  
> **创建日期**: 2026-03-20  
> **目标引擎**: Godot 4.x  
> **渲染器**: Forward+ (推荐) / Mobile (备选)

---

## 一、渲染方案概述

### 1.1 渲染器选择

| 渲染器 | 适用场景 | 支持特性 | 限制 |
|--------|----------|----------|------|
| **Forward+** | 推荐方案 | 多光源、SCREEN_TEXTURE、DEPTH_TEXTURE | 需要DX12/Vulkan支持 |
| **Mobile** | 低端配置 | 基础功能、性能优化 | 部分后处理受限 |
| **Compatibility** | 不推荐 | 最广泛支持 | 无SCREEN_TEXTURE、无DEPTH_TEXTURE |

**决策**: 使用 Forward+ 渲染器，目标帧率 60fps。

### 1.2 美术风格技术实现

- **Low Poly**: 使用低面数模型 + 平面着色
- **明亮卡通色块**: 使用 `render_mode unshaded` 或自定义光照计算
- **发光效果**: 使用 `EMISSION` 输出而非后处理Bloom（性能更好）

#### 3.2.2 Uniform 提示规范

| 参数类型 | 提示语法 | 用途 |
|----------|----------|------|
| 颜色 | `: source_color` | 基础颜色、发光颜色 |
| 颜色（HDR） | `: source_color, hint_color` | HDR发光颜色 |
| 浮点范围 | `: hint_range(min, max)` | 强度、速度等 |
| 浮点范围（步进） | `: hint_range(min, max, step)` | 需要精确控制的值 |
| 纹理 | `: source_color` | 基础贴图 |
| 法线贴图 | `: hint_normal` | 法线纹理 |
| 默认白色 | `: hint_default_white` | 可选遮罩/噪声 |
| 默认黑色 | `: hint_default_black` | 可选遮罩 |

#### 3.2.3 命名规范

- **文件命名**: 小写下划线，如 `fragile_glow.gdshader`
- **Uniform命名**: 小写下划线，如 `emission_color`, `pulse_speed`
- **函数命名**: 小写下划线，如 `calculate_glow()`, `remap_value()`
- **常量命名**: 大写下划线，如 `PI`, `MAX_ITERATIONS`

## 四、MVP阶段着色器实现

### 4.1 易碎品发光效果 (fragile_glow.gdshader)

```glsl
// 文件名: fragile_glow.gdshader
// 描述: 易碎品包裹的淡蓝色微光效果，带有轻微闪烁
// 渲染器要求: Forward+ / Mobile / Compatibility
// 性能预算: 1 纹理采样/片段

shader_type canvas_item;
render_mode blend_mix;

// 颜色参数
uniform vec4 base_color : source_color = vec4(1.0, 1.0, 1.0, 1.0);
uniform vec4 glow_color : source_color = vec4(0.6, 0.8, 1.0, 0.5);

// 发光参数
uniform float glow_intensity : hint_range(0.0, 2.0) = 0.6;
uniform float flicker_speed : hint_range(0.0, 5.0) = 1.5;
uniform float flicker_amount : hint_range(0.0, 0.5) = 0.15;

// 边缘发光参数
uniform float edge_glow_width : hint_range(0.0, 0.5) = 0.1;
uniform float edge_glow_intensity : hint_range(0.0, 2.0) = 1.0;

void fragment() {
    vec4 tex_color = texture(TEXTURE, UV);
    
    // 计算闪烁
    float flicker = sin(TIME * flicker_speed * 6.28318) * 0.5 + 0.5;
    float flicker_multiplier = 1.0 - flicker_amount + flicker * flicker_amount;
    
    // 计算边缘发光（基于alpha通道）
    float alpha = tex_color.a;
    float edge = smoothstep(0.0, edge_glow_width, alpha) - smoothstep(1.0 - edge_glow_width, 1.0, alpha);
    
    // 混合颜色
    vec3 final_color = tex_color.rgb * base_color.rgb;
    vec3 glow = glow_color.rgb * glow_intensity * flicker_multiplier;
    
    // 添加边缘发光
    final_color += glow * edge * edge_glow_intensity;
    
    // 整体微光叠加
    final_color += glow * 0.3;
    
    COLOR = vec4(final_color, tex_color.a * base_color.a);
}
```

**使用说明**:
1. 将着色器应用到易碎品包裹的 Sprite3D 或 MeshInstance 材质
2. 调整 `glow_color` 改变发光颜色（默认淡蓝色）
3. 调整 `flicker_speed` 控制闪烁频率

### 4.2 急件脉冲效果 (urgent_pulse.gdshader)

```glsl
// 文件名: urgent_pulse.gdshader
// 描述: 急件包裹的红色脉冲发光效果
// 渲染器要求: Forward+ / Mobile / Compatibility
// 性能预算: 1 纹理采样/片段

shader_type canvas_item;
render_mode blend_mix;

// 颜色参数
uniform vec4 base_color : source_color = vec4(1.0, 1.0, 1.0, 1.0);
uniform vec4 pulse_color : source_color = vec4(1.0, 0.2, 0.1, 1.0);

// 脉冲参数
uniform float pulse_speed : hint_range(0.5, 5.0) = 2.0;
uniform float pulse_intensity : hint_range(0.0, 3.0) = 1.5;
uniform float pulse_min : hint_range(0.0, 1.0) = 0.3;
uniform float pulse_max : hint_range(0.0, 2.0) = 1.2;

// 外发光参数
uniform float outer_glow_size : hint_range(0.0, 0.3) = 0.08;
uniform float outer_glow_intensity : hint_range(0.0, 2.0) = 0.8;

void fragment() {
    vec4 tex_color = texture(TEXTURE, UV);
    
    // 计算脉冲（正弦波）
    float pulse = sin(TIME * pulse_speed * 6.28318) * 0.5 + 0.5;
    float pulse_value = mix(pulse_min, pulse_max, pulse);
    
    // 基础颜色
    vec3 final_color = tex_color.rgb * base_color.rgb;
    
    // 添加强烈脉冲发光
    vec3 glow = pulse_color.rgb * pulse_intensity * pulse_value;
    final_color += glow;
    
    // 计算外发光（基于距离中心的距离）
    vec2 center = vec2(0.5, 0.5);
    float dist = length(UV - center);
    float outer_glow = smoothstep(0.5, 0.5 - outer_glow_size, dist) * (1.0 - tex_color.a);
    
    // 添加外发光
    final_color += pulse_color.rgb * outer_glow * outer_glow_intensity * pulse_value;
    
    // 增强整体亮度
    final_color *= 1.0 + pulse * 0.3;
    
    COLOR = vec4(final_color, max(tex_color.a, outer_glow * 0.5) * base_color.a);
}
```

**使用说明**:
1. 将着色器应用到急件包裹的材质
2. `pulse_speed` 控制脉冲频率（急件应该更快）
3. `pulse_color` 默认红色，可调整为其他警示色

## 五、第一阶段着色器实现

### 5.1 停电遮罩效果 (blackout_mask.gdshader)

```glsl
// 文件名: blackout_mask.gdshader
// 描述: 停电事件的视野限制效果，圆形遮罩+暗角
// 渲染器要求: Forward+ / Mobile / Compatibility
// 性能预算: 0 纹理采样/片段（纯数学计算）

shader_type canvas_item;
render_mode blend_mix;

// 遮罩参数
uniform float mask_radius : hint_range(0.0, 1.0) = 0.35;
uniform float mask_softness : hint_range(0.0, 0.5) = 0.15;
uniform float darkness : hint_range(0.0, 1.0) = 0.85;

// 暗角参数
uniform float vignette_intensity : hint_range(0.0, 2.0) = 1.2;
uniform float vignette_smoothness : hint_range(0.0, 1.0) = 0.3;

// 噪声参数（可选）
uniform float noise_intensity : hint_range(0.0, 0.2) = 0.05;
uniform float noise_speed : hint_range(0.0, 10.0) = 3.0;

// 简单伪随机函数
float random(vec2 st) {
    return fract(sin(dot(st.xy, vec2(12.9898, 78.233))) * 43758.5453123);
}

void fragment() {
    vec2 center = vec2(0.5, 0.5);
    vec2 uv = UV;
    
    // 计算到中心的距离（考虑屏幕比例）
    float dist = length(uv - center);
    
    // 圆形遮罩
    float mask = smoothstep(mask_radius + mask_softness, mask_radius - mask_softness, dist);
    
    // 暗角效果
    float vignette = smoothstep(1.0, 1.0 - vignette_smoothness, dist * vignette_intensity);
    
    // 噪声（模拟胶片颗粒）
    float noise = random(uv + vec2(TIME * noise_speed)) * noise_intensity;
    
    // 计算最终暗度
    float final_darkness = (1.0 - mask) * darkness + (1.0 - vignette) * 0.3 + noise;
    final_darkness = clamp(final_darkness, 0.0, 1.0);
    
    // 输出：黑色遮罩，中心透明
    COLOR = vec4(0.0, 0.0, 0.0, final_darkness);
}
```

**使用说明**:
1. 创建一个全屏 ColorRect 节点
2. 将此着色器应用到 ColorRect 的材质
3. 确保 ColorRect 的 `mouse_filter` 设置为 `Ignore` 不影响点击
4. 通过动画或代码调整 `mask_radius` 实现视野变化

### 5.2 传送带流动效果 (conveyor_flow.gdshader)

```glsl
// 文件名: conveyor_flow.gdshader
// 描述: 传送带表面的流动效果
// 渲染器要求: Forward+ / Mobile / Compatibility
// 性能预算: 1-2 纹理采样/片段

shader_type spatial;
render_mode blend_mix, depth_draw_opaque, cull_back;

// 基础纹理
uniform sampler2D albedo_texture : source_color;
uniform vec4 base_color : source_color = vec4(0.4, 0.4, 0.45, 1.0);

// 流动参数
uniform float flow_speed : hint_range(0.0, 5.0) = 1.0;
uniform float flow_direction : hint_range(0.0, 6.28318) = 0.0; // 弧度

// 速度线参数
uniform float line_count : hint_range(1.0, 20.0) = 8.0;
uniform float line_width : hint_range(0.01, 0.2) = 0.05;
uniform float line_intensity : hint_range(0.0, 1.0) = 0.3;
uniform vec4 line_color : source_color = vec4(0.6, 0.6, 0.65, 1.0);

// 噪声纹理（用于细节）
uniform sampler2D noise_texture : hint_default_white;
uniform float noise_scale : hint_range(1.0, 50.0) = 20.0;
uniform float noise_intensity : hint_range(0.0, 0.5) = 0.1;

void fragment() {
    // 计算流动方向
    vec2 flow_dir = vec2(cos(flow_direction), sin(flow_direction));
    
    // 流动UV
    vec2 flow_uv = UV + flow_dir * TIME * flow_speed;
    
    // 采样基础纹理
    vec4 albedo = texture(albedo_texture, flow_uv) * base_color;
    
    // 计算速度线
    float line_pos = fract(dot(UV, flow_dir) * line_count + TIME * flow_speed);
    float line = smoothstep(0.0, line_width, line_pos) * smoothstep(line_width * 2.0, line_width, line_pos);
    
    // 添加噪声细节
    float noise = texture(noise_texture, UV * noise_scale).r * noise_intensity;
    
    // 混合颜色
    vec3 final_color = albedo.rgb;
    final_color = mix(final_color, line_color.rgb, line * line_intensity);
    final_color += noise;
    
    ALBEDO = final_color;
    METALLIC = 0.2;
    ROUGHNESS = 0.6;
}
```

**使用说明**:
1. 将着色器应用到传送带 MeshInstance 的材质
2. 调整 `flow_direction` 匹配传送带运动方向
3. 加速事件时，通过代码增加 `flow_speed`

### 5.3 危险品警示效果 (hazard_warning.gdshader)

```glsl
// 文件名: hazard_warning.gdshader
// 描述: 危险品包裹的橙色警告发光效果，带有随机闪烁
// 渲染器要求: Forward+ / Mobile / Compatibility
// 性能预算: 1 纹理采样/片段

shader_type canvas_item;
render_mode blend_mix;

// 颜色参数
uniform vec4 base_color : source_color = vec4(1.0, 1.0, 1.0, 1.0);
uniform vec4 warning_color : source_color = vec4(1.0, 0.5, 0.0, 1.0);
uniform vec4 danger_color : source_color = vec4(1.0, 0.1, 0.0, 1.0);

// 闪烁参数
uniform float flash_speed : hint_range(0.5, 5.0) = 1.8;
uniform float flash_intensity : hint_range(0.0, 2.0) = 1.2;
uniform float randomness : hint_range(0.0, 1.0) = 0.3;

// 条纹参数
uniform float stripe_count : hint_range(2.0, 10.0) = 4.0;
uniform float stripe_width : hint_range(0.1, 0.5) = 0.3;
uniform float stripe_intensity : hint_range(0.0, 1.0) = 0.4;

// 简单伪随机
float random(vec2 st) {
    return fract(sin(dot(st.xy, vec2(12.9898, 78.233))) * 43758.5453123);
}

void fragment() {
    vec4 tex_color = texture(TEXTURE, UV);
    
    // 基础颜色
    vec3 final_color = tex_color.rgb * base_color.rgb;
    
    // 计算闪烁（带随机性）
    float time_var = TIME * flash_speed;
    float flash = sin(time_var * 6.28318) * 0.5 + 0.5;
    float rand = random(vec2(floor(time_var), 0.0)) * randomness;
    flash = mix(flash, rand, randomness);
    
    // 警示条纹（对角线）
    float diagonal = UV.x + UV.y;
    float stripe = step(fract(diagonal * stripe_count), stripe_width);
    
    // 混合警示颜色
    vec3 warning = mix(warning_color.rgb, danger_color.rgb, flash);
    
    // 添加发光
    final_color += warning * flash * flash_intensity;
    
    // 添加条纹
    final_color = mix(final_color, warning, stripe * stripe_intensity * flash);
    
    // 整体亮度脉冲
    final_color *= 1.0 + flash * 0.2;
    
    COLOR = vec4(final_color, tex_color.a * base_color.a);
}
```

## 七、公共函数库

### 7.1 utils.gdshaderinc

```glsl
// 文件名: utils.gdshaderinc
// 描述: 公共着色器函数库

#ifndef UTILS_GDSHADERINC
#define UTILS_GDSHADERINC

// 常量
const float PI = 3.14159265359;
const float TAU = 6.28318530718;

// 将值从 [in_min, in_max] 映射到 [out_min, out_max]
float remap(float value, float in_min, float in_max, float out_min, float out_max) {
    return out_min + (value - in_min) * (out_max - out_min) / (in_max - in_min);
}

// 2D旋转矩阵
mat2 rotate2d(float angle) {
    float c = cos(angle);
    float s = sin(angle);
    return mat2(vec2(c, -s), vec2(s, c));
}

// 简单伪随机
float random(vec2 st) {
    return fract(sin(dot(st.xy, vec2(12.9898, 78.233))) * 43758.5453123);
}

// 2D噪声
float noise(vec2 st) {
    vec2 i = floor(st);
    vec2 f = fract(st);
    
    float a = random(i);
    float b = random(i + vec2(1.0, 0.0));
    float c = random(i + vec2(0.0, 1.0));
    float d = random(i + vec2(1.0, 1.0));
    
    vec2 u = f * f * (3.0 - 2.0 * f);
    
    return mix(a, b, u.x) + (c - a) * u.y * (1.0 - u.x) + (d - b) * u.x * u.y;
}

// 分形布朗运动（FBM）
float fbm(vec2 st, int octaves) {
    float value = 0.0;
    float amplitude = 0.5;
    float frequency = 1.0;
    
    for (int i = 0; i < octaves; i++) {
        value += amplitude * noise(st * frequency);
        amplitude *= 0.5;
        frequency *= 2.0;
    }
    
    return value;
}

// HSV到RGB转换
vec3 hsv2rgb(vec3 c) {
    vec4 K = vec4(1.0, 2.0 / 3.0, 1.0 / 3.0, 3.0);
    vec3 p = abs(fract(c.xxx + K.xyz) * 6.0 - K.www);
    return c.z * mix(K.xxx, clamp(p - K.xxx, 0.0, 1.0), c.y);
}

// RGB到HSV转换
vec3 rgb2hsv(vec3 c) {
    vec4 K = vec4(0.0, -1.0 / 3.0, 2.0 / 3.0, -1.0);
    vec4 p = mix(vec4(c.bg, K.wz), vec4(c.gb, K.xy), step(c.b, c.g));
    vec4 q = mix(vec4(p.xyw, c.r), vec4(c.r, p.yzx), step(p.x, c.r));
    
    float d = q.x - min(q.w, q.y);
    float e = 1.0e-10;
    return vec3(abs(q.z + (q.w - q.y) / (6.0 * d + e)), d / (q.x + e), q.x);
}

#endif // UTILS_GDSHADERINC
```

**使用方式**:
```glsl

```

## 八、性能优化指南

### 8.1 纹理采样预算

| 目标平台 | 每片段最大采样数 | 说明 |
|----------|-----------------|------|
| PC (Forward+) | 8-16 | 宽松限制 |
| PC (Mobile) | 4-8 | 中等限制 |
| 低端设备 | 2-4 | 严格限制 |

### 8.2 避免的性能陷阱

| 陷阱 | 影响 | 解决方案 |
|------|------|----------|
| 每帧动态循环 | 移动端性能差 | 使用固定次数循环或展开 |
| `discard` 在不透明通道 | 破坏Early-Z | 使用 Alpha Scissor |
| 复杂分支 | 寄存器压力 | 使用 `mix()` 或 `step()` |
| `SCREEN_TEXTURE` 过度使用 | 帧缓冲复制 | 合并后处理通道 |

### 8.3 优化检查清单

- [ ] 所有 uniform 都有适当的 hint
- [ ] 没有硬编码的魔法数字
- [ ] 纹理采样数在预算内
- [ ] 移动端没有动态循环
- [ ] 不透明材质避免使用 `discard`
- [ ] 在目标渲染器上测试通过
